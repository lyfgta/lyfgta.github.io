Safari兼容H5播放器

一言加载中...

Safari浏览器是苹果自带的浏览器，和其他浏览器不同的是，它不兼容video标签的Mp4、Webm、Ogg格式，兼容苹果公司推出的M3u8格式视频。下面就教大家如何把Mp4格式转换成M3u8格式视频！

第一步 安装FFmpeg

FFmpeg是一套可以用来记录、转换数字音频、视频，并能将其转化为流的开源计算机程序。

下载地址：

官网下载

解压压缩包，打开目录里的bin文件夹，并复制文件夹地址，右键打开我的电脑/属性/高级系统设置/环境变量/在系统变量里找到Path双击打开，把复制的地址添加进去。

第二步 验证FFmpeg

Win + R 输入 CMD 打开并输入

$ ffmpeg -version

出现版本号

代表安装成功

第三步 转换视频

$ ffmpeg -i E:\demo\test.MP4 -profile:v baseline -level 3.0 -s 640x360 -start_number 0 -hls_time 10 -hls_list_size 0 -f hls E:\demo\test.m3u8

目录地址和文件名称替换即可！

第四步 插入代码

$ source src="//lyfgta.gitee.io/video/xsxzs/test.m3u8" type="application/x-mpegURL"
